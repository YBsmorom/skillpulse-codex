# SkillPulse v0.1.0

这是 SkillPulse 的第一个公开发行版。

## 下载

- `SkillPulse_0.1.0_x64-setup.exe`：推荐使用的 Windows 安装包。
- `SkillPulse_0.1.0_x64_en-US.msi`：MSI 安装包。

## 主要功能

- Windows-only Codex App 悬浮图标；
- 中文可视化面板；
- 本机 Skill 调用统计；
- Skill 库搜索、筛选、排序；
- 趋势、热力图、链路网络；
- 重复、缺说明、吃灰 Skill 等维护提醒；
- 中文备注导出/导入工作流；
- CSV / JSON 报告导出。

## 说明

SkillPulse 是 Codex vibe coding 产物，不是官方 Codex 功能。它默认只读本机 Codex/Agent Skill 与会话索引，不上传数据。

## 已知限制

- 当前只实现 Windows + Codex App 场景；
- 链路统计是基于本机会话记录的推断，不等同于 Codex 官方调用链；
- Skill 启用/停用管理尚未实现；
- 未来 Codex 日志格式变化时可能需要同步调整。
